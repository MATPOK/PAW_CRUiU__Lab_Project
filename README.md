**** Robocze readme *****
1) usunąć zawartość /backend/prisma/migrations/
2) w folderze głównym projektu docker compose up -d
3) cd backend
4) npx prisma migrate dev --name init
5) npm run start:dev
6) w nowej konsoli cd frontend
7) npm run dev
8) aplikacja działa

5173 - FE
3000/api - Swagger
8080 - Adminer
